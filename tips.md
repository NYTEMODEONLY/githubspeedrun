# Tips for GitHub Achievement Speedrunning

## Quickdraw (Fastest)
- Open an issue, immediately close it with a comment
- Works with PRs too

## YOLO (Easy)
- Create a branch on your own repo
- Open a PR and merge without requesting review
- Branch protection rules must not require reviews

## Pull Shark (Grindable)
- Every merged PR counts, even on your own repos
- Default badge at 2 merged PRs
- Bronze at 16 - achievable in a single session

## Galaxy Brain (Requires Discussions)
- Enable Discussions on your repo first
- Create Q&A category discussions
- Post an answer and mark it as accepted
- Need 2 accepted answers for default badge

## Pair Extraordinaire (Needs a Partner)
- Add `Co-authored-by: Name <email>` in commit messages
- The email must map to a real GitHub account
- The PR must be merged

## Starstruck (Community Dependent)
- Build something useful or interesting
- Share on social media, Reddit, Hacker News
- Need 16 stars for default badge

## Public Sponsor (Costs Money)
- Find a developer with low minimum tiers ($1-2/month)
- Sponsor them via GitHub Sponsors
- Badge appears on your profile
