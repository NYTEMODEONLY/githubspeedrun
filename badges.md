# GitHub Profile Badges (Non-Achievement)

Beyond achievements, GitHub profiles can display these special badges:

## Highlight Badges
| Badge | How to Get |
|---|---|
| Pro | Subscribe to GitHub Pro ($4/month) |
| Developer Program Member | Register at github.com/developer |
| Security Bug Bounty Hunter | Participate in GitHub Security bounty |
| Campus Expert | Join GitHub Campus Expert program (students) |
| Security Advisory Credit | Submit accepted advisory to GitHub Advisory Database |

## Historical Badges
| Badge | How It Was Earned |
|---|---|

| Mars 2020 Contributor | Contributed to Mars Helicopter Mission repos |
