# Achievement Tier Details

## Tier System

Each tiered achievement progresses through 4 levels:

```
Default -> Bronze (x2) -> Silver (x3) -> Gold (x4)
```

The multiplier indicates how many times the base achievement image
is decorated with additional visual flair.

## Full Tier Table

| Achievement | Default | Bronze | Silver | Gold |
|---|---|---|---|---|
| Pull Shark | 2 PRs | 16 PRs | 128 PRs | 1,024 PRs |
| Galaxy Brain | 2 answers | 8 answers | 16 answers | 32 answers |
| Pair Extraordinaire | 1 co-author PR | 10 | 24 | 48 |
| Starstruck | 16 stars | 128 stars | 512 stars | 4,096 stars |

## Tier Colors

| Tier | Hex Color |
|---|---|
| Bronze | #F9BFA7 |
| Silver | #E1E4E4 |
| Gold | #FAE57E |

## Non-Tiered Achievements

These achievements have only one level:
- Quickdraw
- YOLO
- Public Sponsor
